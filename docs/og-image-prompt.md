# Open Graph Image Prompt

Use the following prompt to generate an Open Graph image (1200x630px) for social media sharing:

---

**Prompt:**

```
A sleek, modern Open Graph image for Sheriax Solutions technology company. Dark gradient background transitioning from deep navy blue (#0a1628) to rich purple (#2d1b4e). 

Center: The text "Sheriax Solutions" in clean, bold white sans-serif typography (Geist or Inter style). Below it, the tagline "Technology beyond time" in lighter, smaller text.

Subtle decorative elements: Floating holographic geometric shapes (hexagons, circles, connecting lines) representing AI, web, mobile, and VR technology. Soft blue and purple glowing accents.

Clean, professional, premium tech aesthetic. No clutter. High contrast for social media visibility. 1200x630 pixels.
```

---

**Alternative prompt for generated OG image:**

```
Premium technology company Open Graph banner, 1200x630px. Dark space-like gradient background with subtle star particles. Central focus: "Sheriax" wordmark in futuristic white typography with subtle blue glow. Abstract neural network pattern radiating outward. Icons representing AI brain, globe, smartphone, magnifying glass, and VR headset floating elegantly. Modern glass-morphism style elements. Color palette: deep navy, electric blue, soft purple accents. Professional, innovative, trustworthy aesthetic.
```

---

## Usage

After generating the image:
1. Save as `og-image.jpg` (or `.png` for better quality)
2. Place in `/public/`
3. Update metadata in `src/app/layout.tsx` to reference the correct filename
